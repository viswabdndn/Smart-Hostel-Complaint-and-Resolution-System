# 🏨 Smart Hostel Complaint & Resolution System

A web-based application designed to manage hostel complaints efficiently.
Students can raise complaints, and admins can view, update, and resolve them in a structured way.

---

## 🚀 Features

* 👨‍🎓 Student Registration & Login
* 📝 Raise Complaints
* 📋 View Complaint Status
* 🛠️ Admin Dashboard
* ✅ Update & Resolve Complaints
* 💾 File Handling for Data Storage
* 🔄 Auto-save functionality

---

## 🛠️ Technologies Used

* **Java**
* **Spring Boot**
* **HTML, CSS**
* **File Handling (Text-based storage)**
* **Maven**
